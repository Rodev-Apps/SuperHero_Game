Here's how to play the coolest game in town.

1. Player(user) and computer have an equal number of cards in the start of the game.
2. User will start the game by default. n equal number of cards in the start of the game.
3. There are 6 stats with a corresponding value: Combat, Durability, Intelligence, Power, Speed and Strength n equal number of cards in the start of the game.
4. User can view their card but cannot see the computers until they have chosen the stats to battle. n equal number of cards in the start of the game.
5. User should choose 1 out of 6 stats on the card, if the value of the stat is higher than the value of stat from the computer's card, the user will win the round n equal number of cards in the start of the game.
6. Vice versa, Computer will win if it's stats is higher n equal number of cards in the start of the game.
7. you loose a card if you loose a round and your opponent gains that card n equal number of cards in the start of the game.
8. The game ends when one person runs out of cards,and the player with all the cards wins
